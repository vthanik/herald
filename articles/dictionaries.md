# Controlled Terminology and Dictionary Providers

Clinical validation depends on reference data: CDISC controlled
terminology, SRS, MedDRA, WHODrug, LOINC, SNOMED, and sponsor
dictionaries. `herald` uses one interface for all of them: a dictionary
provider.

## Registering providers

Register providers globally for the R session, or pass them directly to
`validate(dictionaries = ...)` for one run.

``` r
register_dictionary("ct-sdtm", ct_provider("sdtm"))
list_dictionaries()
#> # A tibble: 1 × 5
#>   name    version    source  license   size_rows
#>   <chr>   <chr>      <chr>   <chr>         <int>
#> 1 ct-sdtm 2026-03-27 bundled CC-BY-4.0     45612
unregister_dictionary("ct-sdtm")
```

One-run registration avoids session state:

``` r
dm <- data.frame(USUBJID = "S1", stringsAsFactors = FALSE)

result <- validate(
  files = dm,
  rules = character(0),
  dictionaries = list("ct-sdtm" = ct_provider("sdtm")),
  quiet = TRUE
)

result
#> ── herald validation -- Spec Checks Only ──────────────────────────────
#> Rules: 0/0 applied
#> Datasets checked: 1
#> Findings: 0
#> Duration: 0.02690029 secs
```

## Custom dictionaries

Use
[`custom_provider()`](https://vthanik.github.io/herald/reference/custom_provider.md)
when your dictionary is already in a data frame.

``` r
sponsor_terms <- data.frame(
  VISIT = c("SCREENING", "BASELINE", "WEEK 1"),
  stringsAsFactors = FALSE
)

visit_provider <- custom_provider(
  sponsor_terms,
  name = "sponsor-visits",
  fields = "VISIT"
)

visit_provider
#> <herald_dict_provider>
#>   name     : sponsor-visits
#>   version  : unknown
#>   source   : sponsor
#>   license  : sponsor-private
#>   size     : 3 rows
#>   fields   : VISIT
visit_provider$contains(c("BASELINE", "EARLY TERMINATION"), field = "VISIT")
#> [1]  TRUE FALSE
```

For full control, build a provider directly with
[`new_dict_provider()`](https://vthanik.github.io/herald/reference/new_dict_provider.md).

``` r
allowed <- c("LOW", "MEDIUM", "HIGH")

risk_provider <- new_dict_provider(
  name      = "risk-levels",
  version   = "2026-04",
  source    = "sponsor",
  license   = "internal",
  size_rows = length(allowed),
  fields    = "RISK",
  contains  = function(value, field = NULL, ignore_case = FALSE) {
    value <- as.character(value)
    if (isTRUE(ignore_case)) {
      return(toupper(value) %in% toupper(allowed))
    }
    value %in% allowed
  }
)

risk_provider$contains(c("LOW", "UNKNOWN"), field = "RISK")
#> [1]  TRUE FALSE
```
